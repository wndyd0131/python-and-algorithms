선택 정렬
- 가장 작은 값을 선택해서 앞으로 보내는 방식
- O(N^2)
- 구현 방법: 가장 작은 값의 인덱스를 구해서 i번째랑 스왑
- POINT: min값 찾을 때 사용하면 괜찮음

삽입 정렬
- 적절한 위치에 값을 넣어 정렬하는 방식
- 구현 방법: 2번째 원소부터 시작해서 왼쪽을 향해 자신보다 작은 값과 비교해서 스왑
- O(N^2) ~ O(N)
- POINT: 정렬이 되어있을 수록 성능에 좋음

퀵 정렬
- 설정된 피벗의 기준으로 큰수와 작은수를 교환하고 리스트를 나눔
1.
    - 설정된 피벗을 기준으로 좌측부터 피벗보다 큰 값 우측부터 피벗보다 작은 값을 찾아서 교환 
    - 두 값이 엇갈린 경우 작은 값을 피벗과 교환
    - 이때 피벗의 좌측은 작은 값, 우측은 큰 값으로 분할됨
2. 좌측에 대해 반복
3. 우측에 대해 반복
- 구현 방법: 재귀 -> 리스트의 원소가 1개면 스톱 -> left, pivot, right
- O(N^2) ~ O(NlogN)
    - 정렬이 되어있을 수록 성능 저하

계수 정렬
- 범위가 제한 된 데이터에 대해 정렬할 때, 그 범위에 대한 리스트로 각각의 데이터를 카운트하여 해당하는 인덱스에 넣어 출력하는 방식
- O(N+K) N=데이터 개수 K=범위의 최댓값
- POINT: 동일한 값을 가지는 데이터 여러 개 등장할 때 적합 (성적, 키와 같이 통계적인 부분)
    - 데이터 개수가 1,000,000개 미만일 때 효과적

파이썬 정렬 라이브러리
- sort() 또는 sorted()의 매개변수로 key를 넣을 수 있음
    - key는 정렬 기준을 반환하는 함수여야 함
